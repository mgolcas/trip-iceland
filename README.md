# Islandija 2026 🇮🇸

Pietinės pakrantės kelionė nuomotu automobiliu: **2026.09.03 – 2026.09.07** (5 dienos, 3 keliautojai).
Skrydis Vilnius → KEF (Keflavík), maršrutas Ring Road (Kelias 1) palei pietinę pakrantę,
daug gamtos, kriokliai ir 2–3 val. žygis (Waterfall Way).

Automobilis: **Zero Car – Dacia Duster 4WD, mechaninis (rezervuota)**.

## Žemėlapių importas

Importuok `Iceland.kml` ir `Iceland_Dienos.kml` į [Google My Maps](https://www.google.com/mymaps)
kaip atskirus žemėlapius.

## Maršruto santrauka

- **D01 09.03** – Atvykimas + Auksinis ratas (Þingvellir, Geysir, Gullfoss, Kerið) → Hvolsvöllur
- **D02 09.04** – Kriokliai + Waterfall Way žygis (Seljalandsfoss, Skógafoss, Kvernufoss) → Vík
- **D03 09.05** – Sólheimajökull, Dyrhólaey, Reynisfjara → Vík
- **D04 09.06** – Jökulsárlón ledynų lagūna + Fjaðrárgljúfur + Diamond Beach ⚠️ ilga diena → Vík
- **D05 09.07** – Reykjavík + auto grąžinimas + išvykimas
